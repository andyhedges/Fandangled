Building Fandangled

TL;DR

The 'too long; didn't read' version for the incurrably impatient

You need:

  • A copy of the source code, we'll assume you use wget to download but you can use your browser if you prefer
  • A tool to unzip that, we'll assume gnu zip
  • A JDK 1.6 or higher
  • Maven 2 (haven't tried 3)

To build run the following commands

$ wget -O fandangled.zip --no-check-certificate https://github.com/andyhedges/Fandangled/zipball/v0.0.2 $ unzip fandangled.zip $ cd andyhedges-Fandangled-29a1926 $ mvn clean install

Once maven has finished you should have a compiled set of jar files

Unless you are a library user you will be interested in either of the two executable files, one is a gui the other is command line. The commandline is at the following path:

command-line/target/command-line-0.0.2-SNAPSHOT.jar

The gui is at the following path

basic-gui/target/basic-gui-0.0.2-SNAPSHOT.jar

I recommend you use the command line, the GUI was a bit of an experiement and gui design isn't my strength. If you fancy writing a better one please get in touch (tmp1@hedges.net put 'Fandangled' in the subject line somewhere)
